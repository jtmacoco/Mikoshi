- There are 2 categories of the Objective Function: Loss and Reward
## Loss functions
- The lower Loss function, the better the accuracy 
- Usually used in supervised learning
- This is just a function that has a higher value for worse results and vice versa
- So this loss function takes in
	- The output of the model's values i.e. the predicted values
	- The target values i.e. the labeled data (not in RL this is based on reward function)
	- Then the loss compares these 2 values 
	- We talked about different loss function such as
		- L2-norm Loss
		- Cross-Entropy
## Reward Functions
- Used in Reinforcement Learning
- Is the exact opposite of the loss function because we want to maximize reward
